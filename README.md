# eltex

1) Чтобы сгенерировать документацию, перейдите в папку /eltex и пропишите: doxygen Doxyfile
    - откройте в браузере файл html/index.html

2) Чтобы собрать какое-либо из ДЗ, перейдите в родительскую папку ДЗ. Там вы найдете Makefile.
    - команда [make] -> создает бинарник и объектные файлы;
    - команда [make clean] -> удаляет бинарник и объектные файлы.
